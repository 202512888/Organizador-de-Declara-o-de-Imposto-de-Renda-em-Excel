# Organizador-de-Declara-o-de-Imposto-de-Renda-em-Excel
ste projeto foi desenvolvido como desafio da DIO para criar uma ferramenta em Excel que organiza informações essenciais para a declaração do Imposto de Renda.  A planilha permite registrar rendimentos, despesas dedutíveis e gerar um resumo automático dos dados.
